## Installation

### Install using Contao Manager

Search for **ai**, **chatgpt**, or **openai** and you will find this extension.

### Install using Composer

```bash
composer require plenta/loki-ai-bundle
```


## System requirements

- PHP: `^8.3`
- Contao: `^4.13` || `^5`
